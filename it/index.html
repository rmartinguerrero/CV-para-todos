Domande frequenti sul CV builder

È veramente gratuito?
Sì, completamente gratuito. Nessun abbonamento, nessuna pubblicità, nessun piano a pagamento. Il codice è aperto e il tuo CV è ospitato su GitHub Pages, un servizio gratuito di GitHub.
Devo saper programmare per usarlo?
No. L'editor è 100% No-Code: basta compilare i moduli con i tuoi dati e il CV viene generato automaticamente. Non serve scrivere una riga di codice.
Dove vengono archiviati i dati del mio CV?
Nel tuo repository personale di GitHub. Solo tu hai accesso in scrittura. L'app può solo creare e aggiornare file in quel repository specifico.
Posso eliminare il mio CV dopo la pubblicazione?
Sì. Puoi cancellare il repository da GitHub in qualsiasi momento. Il tuo CV smetterà di essere disponibile immediatamente.
Posso modificare il mio CV dopo la pubblicazione?
Sì. Torna all'editor, modifica i tuoi dati e ripubblica. Il tuo sito web si aggiorna istantaneamente mantenendo lo stesso URL.

📥 Consigliato: Carica un file resume.it.json

Puoi scaricare il tuo CV attuale in JSON da qualsiasi editor e reimportarlo qui.
