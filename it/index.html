Domande frequenti sul CV builder

È veramente gratuito?
Sì. Completamente gratuito. Nessun abbonamento, nessuna pubblicità, nessun piano a pagamento. Il codice è aperto e il CV è ospitato su GitHub Pages, un servizio gratuito di GitHub.
Devo saper programmare?
No. L'editor è al 100% visuale. Compili i moduli con i tuoi dati e il CV viene generato automaticamente. Non devi toccare una riga di codice.
Che cos'è JSON Resume?
È un formato aperto e standard per i curriculum. Permette che i tuoi dati vivano separati dal design, cosicché tu possa cambiare modello senza perdere nulla.
Perché usate GitHub?
GitHub Pages è gratuito, affidabile e non dipende da noi. I tuoi dati vivono nel tuo account. Se l'app smettesse di esistere, il tuo CV rimarrebbe disponibile.
Posso cambiare il design dopo?
Sì. Puoi cambiare modello in qualsiasi momento. I tuoi dati rimangono intatti. Cambia solo il design.
Posso scaricare il CV in PDF?
Sì. Dal browser puoi stampare o salvare il tuo CV come PDF con un solo clic.
Dove vivono i miei dati?
Nel tuo repository personale di GitHub. Solo tu hai accesso in scrittura. L'app non ha server propri dove archiviare i tuoi dati.
Cosa succede se smetto di usare l'app?
Nulla. I tuoi dati sono nel tuo account GitHub. Puoi scaricare il JSON, usare un altro editor compatibile con JSON Resume o continuare ad aggiornare manualmente.
Posso importare un resume.json?
Sì. Se hai già un file resume.json, puoi importarlo direttamente e iniziare a modificare.
Posso modificare il CV dopo la pubblicazione?
Sì. Torna all'editor, modifica quello che serve e ripubblica. Il sito si aggiorna istantaneamente.
Posso eliminare il mio CV?
Sì. Cancelli il tuo repository GitHub e il tuo CV smette di essere disponibile immediatamente.
Posso avere più lingue?
Sì. Puoi creare il tuo CV in spagnolo, italiano o inglese. Cambia lingua istantaneamente.
Devo installare qualcosa?
No. Funziona direttamente nel browser. Non devi scaricare né installare nessun programma.
Cosa succede se GitHub cambia?
Tu hai sempre una copia dei tuoi dati in formato standard. Puoi migrare a qualsiasi servizio compatibile con JSON Resume.
Perché è meglio di un editor tradizionale?
Perché separa i dati dal design. In Word o Canva, ogni modifica può rompere il formato. Qui, cambi quello che vuoi e il design si mantiene sempre.
