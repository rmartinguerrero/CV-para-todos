Informativa sulla Privacy

CV para Todos è progettato con un modello “local-first”. Non memorizziamo dati personali sui server e non abbiamo accesso ai contenuti creati nella piattaforma.

Dove vengono salvati i dati

Tutti i dati del CV sono salvati esclusivamente nel tuo browser (LocalStorage / IndexedDB). Non lasciano il tuo dispositivo.

OAuth / autenticación

Se utilizzi login con provider esterni, vengono usati solo token di sessione temporanei. Non memorizziamo credenziali o password.

Google Analytics

Google Analytics si attiva solo dopo il tuo consenso esplicito tramite il banner cookie. I dati sono aggregati e anonimi.

Come vedere i tuoi dati nel browser

Puoi vedere ed eliminare i dati locali nel browser seguendo questi passaggi:

  • Chrome: Impostazioni → Privacy e sicurezza → Cookie → Visualizza dati sito.
  • Firefox: Impostazioni → Privacy → Cookie e dati sito → Gestisci dati.
  • Edge: Impostazioni → Cookie e permessi sito → Visualizza cookie.
  • Safari: Preferenze → Privacy → Gestisci dati siti web.

Controllo dell’utente

Hai pieno controllo sui tuoi dati. Puoi eliminarli in qualsiasi momento dal browser. Non esiste un database centrale.
