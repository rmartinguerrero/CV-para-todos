Frequently asked questions about the CV builder

Is it really free?
Yes, completely free. No subscriptions, no ads, no paid plans. The code is open source and your CV is hosted on GitHub Pages, a free service from GitHub.
Do I need coding skills?
No. The editor is 100% No-Code: just fill in forms with your data and the CV is generated automatically. No need to write a single line of code.
Where is my CV data stored?
In your personal GitHub repository. Only you have write access. The app can only create and update files in that specific repository.
Can I delete my CV after publishing?
Yes. You can delete the repository from GitHub at any time. Your CV will stop being available immediately.
Can I edit my CV after publishing?
Yes. Return to the editor, update your data and publish again. Your website updates instantly while keeping the same URL.
