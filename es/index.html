Preguntas frecuentes sobre el CV builder

¿Es realmente gratis?
Sí. Completamente gratis. Sin suscripciones, sin anuncios, sin planes de pago. El código es abierto y el CV se aloja en GitHub Pages, un servicio gratuito de GitHub.
¿Necesito saber programar?
No. El editor es 100% visual. Rellenas formularios con tus datos y el CV se genera automáticamente. No necesitas tocar una línea de código.
¿Qué es JSON Resume?
Es un formato abierto y estándar para currículums. Permite que tus datos vivan separados del diseño, para que puedas cambiar de plantilla sin perder nada.
¿Por qué usáis GitHub?
GitHub Pages es gratuito, fiable y no depende de nosotros. Tus datos viven en tu propia cuenta. Si la aplicación dejara de existir, tu CV seguiría disponible.
¿Puedo cambiar el diseño después?
Sí. Puedes cambiar de plantilla en cualquier momento. Tus datos se mantienen intactos. Solo cambia el diseño.
¿Puedo descargar mi CV en PDF?
Sí. Desde el navegador puedes imprimir o guardar tu CV como PDF con un solo clic.
¿Dónde viven mis datos?
En tu repositorio personal de GitHub. Solo tú tienes acceso de escritura. La aplicación no tiene servidores propios donde guardar tus datos.
¿Qué pasa si dejo de usar la aplicación?
Nada. Tus datos están en tu cuenta de GitHub. Puedes descargar el JSON, usar otro editor compatible con JSON Resume o seguir actualizando manualmente.
¿Puedo importar un resume.json?
Sí. Si ya tienes un archivo resume.json, puedes importarlo directamente y empezar a editar.
¿Puedo editar el CV después de publicarlo?
Sí. Vuelve al editor, modifica lo que necesites y vuelve a publicar. Tu web se actualiza al instante.
¿Puedo eliminar mi CV?
Sí. Borras tu repositorio de GitHub y tu CV deja de estar disponible inmediatamente.
¿Puedo tener varios idiomas?
Sí. Puedes crear tu CV en español, italiano o inglés. Cambia de idioma al instante.
¿Necesito instalar algo?
No. Funciona directamente en el navegador. No necesitas descargar ni instalar ningún programa.
¿Qué pasa si GitHub cambia?
Tú siempre tienes una copia de tus datos en formato estándar. Puedes migrar a cualquier servicio compatible con JSON Resume.
¿Por qué es mejor que un editor tradicional?
Porque separa los datos del diseño. En Word o Canva, cada cambio puede romper el formato. Aquí, cambias lo que quieras y el diseño siempre se mantiene.
