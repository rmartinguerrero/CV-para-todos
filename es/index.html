Preguntas frecuentes sobre el CV builder

¿Es realmente gratis?
Sí, completamente gratis. Sin suscripciones, sin anuncios, sin planes de pago. El código es abierto y tu CV se aloja en GitHub Pages, un servicio gratuito de GitHub.
¿Necesito saber programar para usarlo?
No. El editor es 100% No-Code: simplemente rellenas formularios con tus datos y el CV se genera automáticamente. No necesitas tocar una línea de código.
¿Dónde se almacena la información de mi CV?
En tu repositorio personal de GitHub. Solo tú tienes acceso de escritura. La aplicación solo puede crear y actualizar archivos en ese repositorio específico.
¿Puedo eliminar mi CV después de publicarlo?
Sí. Puedes borrar el repositorio desde GitHub en cualquier momento. Tu CV dejará de estar disponible inmediatamente al eliminar el repositorio.
¿Puedo editar mi CV después de publicarlo?
Sí. Vuelve al editor, modifica tus datos y vuelve a publicar. Tu sitio web se actualiza al instante conservando la misma URL.

📥 Recomendado: Sube un archivo resume.es.json

Puedes descargar tu CV actual en JSON desde cualquier editor y reimportarlo aquí.
